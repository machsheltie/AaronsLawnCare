## Image Specifications

### Open Graph Images (for Facebook, LinkedIn, Pinterest)

**Filename Format:** `og-{page-slug}.jpg`

**Specifications:**
- **Dimensions:** 1200 x 630 pixels
- **Aspect Ratio:** 1.91:1 (horizontal, landscape)
- **Format:** JPG or PNG
- **Max File Size:** 500 KB
- **Color Mode:** RGB (not CMYK)
- **DPI:** 72 DPI (web standard)

**Best Practices:**
- Use high contrast and readable text
- Include company logo/branding
- Place important elements in center (safe zone: 100-1100px horizontal, 130-500px vertical)
- Test with 8:9 mobile view (will be cropped)
- Use clean, professional design
- Include a call-to-action

**Examples:**
- `og-home.jpg` - Hero image of lawn, company branding
- `og-service-lawn-mowing.jpg` - Before/after lawn mowing
- `og-about.jpg` - Team photo or company building
- `og-contact.jpg` - Office location or contact info
- `og-blog-spring-tips.jpg` - Relevant lawn image with title

### Twitter Card Images (for Twitter/X)

**Filename Format:** `twitter-{page-slug}.jpg`

**Specifications:**
- **Dimensions:** 1200 x 675 pixels
- **Aspect Ratio:** 16:9 (horizontal, widescreen)
- **Format:** JPG or PNG
- **Max File Size:** 500 KB
- **Color Mode:** RGB (not CMYK)
- **DPI:** 72 DPI (web standard)

**Best Practices:**
- Design for 16:9 aspect ratio (wider than OG images)
- Keep important content in center 50% of image
- Use clear, bold visuals
- Ensure text is readable at small sizes
- Test on mobile phones
- Avoid overly complex designs

**Examples:**
- `twitter-home.jpg` - Landscape lawn scene with company name
- `twitter-service-lawn-mowing.jpg` - Mowing in progress photo
- `twitter-about.jpg` - Team photo or company location
- `twitter-contact.jpg` - Contact information with branding
- `twitter-blog-spring-tips.jpg` - Seasonal lawn image

## Design Recommendations

### Color Scheme
- Use brand colors: Green (#15803d primary, #22c55e secondary)
- Ensure 4.5:1 contrast ratio for text readability
- Avoid pure black/white (use charcoal/off-white)

### Typography
- Use readable fonts (Sans-serif recommended)
- Min font size: 24pt for web viewing
- Keep text to 1-3 lines maximum
- Place text in image center area

### Images & Photos
- Use high-quality lawn/landscaping photos
- Include before/after comparisons when relevant
- Maintain consistent branding across images
- Consider seasonal variations

## File Optimization

### Before Upload

1. **Reduce File Size**
   - Target: 300-500 KB per image
   - Use compression tools: TinyPNG, ImageOptim, ImageMagick

2. **Verify Dimensions**
   - Use image info tools or Photoshop
   - Ensure exact dimensions (not approximately)

3. **Convert Format**
   - JPG for photos and complex images
   - PNG for graphics/designs with transparency
   - Use JPG for smaller file sizes

4. **Remove Metadata**
   - Remove EXIF data with ExifTool
   - Prevents tracking information leakage

### Tools for Optimization
- **TinyPNG/TinyJPG:** https://tinypng.com/
- **ImageOptim:** https://imageoptim.com/
- **ExifTool:** https://exiftool.org/
- **FFmpeg:** For batch processing

## URL Structure in Code

When referencing images in `seo-meta.ts`:

```tsx
const ogImageUrl = `${BASE_URL}${IMAGE_PATH}/og-home.jpg`;
// Results in: https://aaronslawncare.com/images/og-home.jpg

const twitterImageUrl = `${BASE_URL}${IMAGE_PATH}/twitter-home.jpg`;
// Results in: https://aaronslawncare.com/images/twitter-home.jpg
```

**Note:** BASE_URL = "https://aaronslawncare.com" and IMAGE_PATH = "/images"

## Performance Considerations

### Image Delivery
- Use CDN for faster image delivery (optional)
- Images don't affect page load (separate requests)
- Keep file sizes under 500 KB

### Caching
- Social platforms cache images for 24-48 hours
- Use Facebook/Twitter debuggers to clear cache
- Test again after 24 hours if images still wrong

## Accessibility

### Alt Text
Always include `ogImageAlt` in SEOHead:

```tsx
<SEOHead
  ogImage="https://aaronslawncare.com/images/og-home.jpg"
  ogImageAlt="Aaron's Lawn Care - Professional lawn with green grass and beautiful landscape"
/>
```

### Best Practices
- Describe image content clearly
- Include brand/service name
- Keep under 125 characters
- Don't start with "image of" (already implied)

## Troubleshooting

### Image Not Showing
1. Verify URL is correct and publicly accessible
2. Test URL in browser (should download image)
3. Check file extension (.jpg, not .JPG on case-sensitive servers)
4. Verify image dimensions are exactly right
5. Wait 24-48 hours for platform cache refresh

### Wrong Image Showing
1. Clear platform cache using debugger
2. Verify URL points to correct image
3. Check filename spelling and location
4. Re-scrape page in platform debugger

### Image Looks Distorted
1. Verify image dimensions are correct
2. Check aspect ratio in platform preview
3. Use safe zone guidelines for text placement
4. Test on mobile view in platform debugger

### File Too Large
1. Compress with TinyPNG/TinyJPG
2. Convert PNG to JPG
3. Reduce resolution (but keep dimensions)
4. Use optimizer tool
